# Bus App

Aplicación de gestión de autobuses construida con React, TypeScript y Vite.

## 🚀 Tecnologías

- **Frontend**: React 19 + TypeScript
- **Build**: Vite
- **Estilos**: Tailwind CSS
- **Router**: React Router DOM
- **Formularios**: React Hook Form

## 🛠️ Scripts

```bash
# Iniciar desarrollo
git clone https://github.com/HottaIssa/bus-app-frontend.git
cd bus-app-frontend
npm install
npm run dev
```

## 🔐 Funcionalidades

- Registro e inicio de sesión
- Protección de rutas
- Listado de autobuses
- Detalle de autobús
- Gestión de autenticación con Context API
